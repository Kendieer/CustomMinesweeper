## Introduction

一个简单的扫雷游戏，其中包含基础的扫雷功能，以及附加的存档读档功能。


除此以外，还有可以读档、存档功能，以便中途退出游戏。

> **关于随机种子**：当使用了固定种子并且首个点击节点相同时，那么生成的地图就相同，若不满足任意条件，则地图不同，随机种子在每局点击第一个节点后才生效，并非创建游戏时。
